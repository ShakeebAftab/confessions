# Confessions

Social Media App which allows you to post Confessions anonymously or with your real name

## Features

- Like/Dislike
- Users can comment on posts
- Working Auth
- Fully Cross platform

## Tech Stack

**Client:** React, Context API

**Server:** Node, Express, Firebase
